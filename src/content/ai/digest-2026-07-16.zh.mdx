## <a href="https://arxiv.org/abs/2607.12625" target="_blank" rel="noopener noreferrer">KnowAct-GUIClaw: Know Deeply, Act Perfectly, Personal GUI Assistant with Self-Evolving Memory and Skill</a>

*HF ★ 7 · Yunxin Li, Jinchao Li, Shibo Su… · [HF 镜像](https://huggingface.co/papers/2607.12625)*

针对主流GUI自动化代理框架OpenClaw跨平台适配差、缺乏自进化机制的缺陷，研究团队提出新型个人GUI助理KnowAct-GUIClaw，采用“知-路-行-思”架构，搭载可进化记忆与技能库，支持四大主流系统。实测其长序列任务精度达64.1%，优于GPT-5.5等闭源模型，技能可跨基座迁移，搭配Kimi-2.6性能再升8.5%。

## <a href="https://arxiv.org/abs/2607.12395" target="_blank" rel="noopener noreferrer">Ring-Zero: Scaling Zero RL to a Trillion Parameters for Emergent Reasoning</a>

*HF ★ 2 · Xinyu Tang, Gangqiang Cao, Yurou Liu… · [HF 镜像](https://huggingface.co/papers/2607.12395)*

针对无需人工标注的零样本强化学习现有方案仅适配小模型、直接缩放存在可读性差、推理冗余等问题，本文提出Ring-Zero训练管线，通过截断重要性采样、训推比校正等优化首次将其拓展至1T参数规模。实验显示缩放可显著提升样本效率与性能上限，模型自发涌现多种高级认知能力，在7个数学基准表现优异，还提出三维思维链评估框架，为大参数量研究提供参考。

## <a href="https://arxiv.org/abs/2607.11896" target="_blank" rel="noopener noreferrer">OmniPMNet: Bridging discrete and gridded PM10 forecasts via omni-query neural processes</a>

*Shuangshuang He, Shuo Wang*

针对PM10预报中化学传输模式局地偏差大、图神经网络仅能输出离散站点结果的痛点，提出融合模型OmniPMNet：基于卷积条件神经过程，将站点预报栅格化后融合CAMS格点预报，可同步输出108小时的站点、网格化PM10结果。经2024年中国1618个站点验证，其站点精度优于基准图神经网络，较CAMS误差降30%，高浓度、沙尘场景提升更显著。（共122字）

*Tiantian Zhang*

## <a href="https://arxiv.org/abs/2607.11928" target="_blank" rel="noopener noreferrer">Repairing Shape-Prior Shortcuts in Long-Range Single-Shot Fringe Projection Profilometry</a>

*Adam Haroon, Cody Fleming, Beiwen Li*

单帧条纹投影轮廓术深度学习模型存在形状先验捷径，依赖物体边界而非条纹相位算深度，仅加数据或模型容量无法破局。本文提出PhiCalNet：输出包裹相位正余弦表征，经固定可微校准层映射深度，引入条纹阶次作辅助输入。在1.5-2.1m远距测试集上，其平均绝对误差较基线降3.3倍至4.46mm。
