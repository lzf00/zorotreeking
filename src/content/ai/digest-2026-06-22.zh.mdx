# Hugging Face Daily Papers

## <a href="https://arxiv.org/abs/2606.19534" target="_blank" rel="noopener noreferrer">PerceptionDLM: Parallel Region Perception with Multimodal Diffusion Language Models</a>

*HF ★ 26 · Yueyi Sun, Yuhao Wang, Jason Li… · [HF 镜像](https://huggingface.co/papers/2606.19534)*

针对现有多模态大模型依赖自回归生成、多区域感知效率低的问题，本文提出多模态扩散语言模型PerceptionDLM，通过高效提示、结构化注意力掩码实现序列与令牌级并行生成多区域描述，同步构建多区域描述评测基准ParaDLC-Bench。实验证明，该模型在保持区域描述性能竞争力的同时大幅提升多区域感知速度，是首个用扩散语言模型实现并行区域感知的方案，相关资源已开源。

## <a href="https://arxiv.org/abs/2606.19651" target="_blank" rel="noopener noreferrer">BrainG3N: A Dual-Purpose Tokenizer for Controllable 3D Brain MRI Generation</a>

*HF ★ 2 · Max Van Puyvelde, Ibrahim Gulluk, Wim Van Criekinge… · [HF 镜像](https://huggingface.co/papers/2606.19651)*

针对3D脑MRI潜扩散现有分词器重重建、丢失临床信息的痛点，本文提出双用途分词器BrainG3N：冻结用3.5万余多模态多疾病样本预训练的3D掩码自编码器编码器提取临床特征，搭配独立CNN解码器完成重建。该模型在23项临床任务中21项超/持平SOTA，同时支持可控生成与病程预测，兼顾下游任务和生成需求。

<FeedbackButtons slug="item:arxiv-2606.19651" client:visible />

## <a href="https://arxiv.org/abs/2606.18847" target="_blank" rel="noopener noreferrer">WorldLines: Benchmarking and Modeling Long-Horizon Stateful Embodied Agents</a>

*HF ★ 2 · Yehang Zhang, Jianchong Su, Haojian Huang… · [HF 镜像](https://huggingface.co/papers/2606.18847)*

针对现有长时记忆基准多侧重语言类问答、具身智能基准多聚焦短任务、未覆盖动态环境长时记忆能力评测的不足，该研究推出WorldLines长时程具身家庭辅助基准，构建长时序家庭交互轨迹生成评测样本，还提出ObsMem记忆框架支撑状态感知决策。实验表明当前方案仍存部分可观测、状态覆写等难点，ObsMem为该场景提供更优参考架构。

<FeedbackButtons slug="item:arxiv-2606.18847" client:visible />

## <a href="https://arxiv.org/abs/2606.18829" target="_blank" rel="noopener noreferrer">GateMem: Benchmarking Memory Governance in Multi-Principal Shared-Memory Agents</a>

*HF ★ 1 · Zhe Ren, Yibo Yang, Yimeng Chen… · [HF 镜像](https://huggingface.co/papers/2606.18829)*

现有大模型智能体内存基准多适配单用户场景，缺乏多主体共享内存场景的测评能力。本文推出覆盖医疗、办公、教育、家居场景的GateMem基准，可同步测评合法请求效用、跨边界访问控制、删除后主动遗忘三类核心能力。测试表明现有方案均无法同时满足三类要求：长上下文方案治理效果好但token成本高，检索类方案成本低却易发生信息泄露，距可靠共享部署仍有较大差距。
