# Hugging Face Daily Papers

## <a href="https://arxiv.org/abs/2606.24530" target="_blank" rel="noopener noreferrer">NatureBench: Can Coding Agents Match the Published SOTA of Nature-Family Papers?</a>

*HF ★ 24 · Yuru Wang, Lejun Cheng, Yuxin Zuo… · [HF 镜像](https://huggingface.co/papers/2606.24530)*

本研究推出跨学科基准NatureBench，含从Nature系列论文提炼的90项任务，配套NatureGym流水线构建标准化容器环境，解决过往基准环境碎片化、可信度低的问题。禁用网络搜索测试10款前沿编码智能体，发现最强模型仅17.8%任务超现有SOTA，成功多靠转成熟悉的监督预测问题而非原创，失败主因是方法选错、算力不足，相关资源已开源。

## <a href="https://arxiv.org/abs/2606.24597" target="_blank" rel="noopener noreferrer">Qwen-AgentWorld: Language World Models for General Agents</a>

*HF ★ 22 · Yuxin Zuo, Zikai Xiao, Li Sheng… · [HF 镜像](https://huggingface.co/papers/2606.24597)*

本文提出面向通用智能体的语言世界模型Qwen-AgentWorld，基于千万级跨7领域真实交互轨迹，经三阶段训练推出两款参数规格模型，配套自研测评基准AgentWorldBench验证，性能显著优于现有前沿模型。同时验证两类应用范式：作为独立模拟器可提升智能体强化学习效果，作为预训练预热方案可提升7类下游智能体任务表现。

<FeedbackButtons slug="item:arxiv-2606.24597" client:visible />

## <a href="https://arxiv.org/abs/2606.19930" target="_blank" rel="noopener noreferrer">MobileForge: Annotation-Free Adaptation for Mobile GUI Agents with Hierarchical Feedback-Guided Policy Optimization</a>

*HF ★ 15 · Guangyi Liu, Pengxiang Zhao, Gao Wu… · [HF 镜像](https://huggingface.co/papers/2606.19930)*

针对基于MLLM的移动端GUI代理适配真实应用标注成本高、无标注学习框架零散、优化信号粗糙的痛点，本文提出无标注适配系统MobileForge：搭载真实应用交互的任务生成评估模块，采用分层反馈引导的策略优化方法，无需人工标注即可完成适配，性能接近闭域专用GUI模型，跨域测试达当前公开方案最优。

<FeedbackButtons slug="item:arxiv-2606.19930" client:visible />

## <a href="https://arxiv.org/abs/2606.19926" target="_blank" rel="noopener noreferrer">MemGUI-Agent: An End-to-End Long-Horizon Mobile GUI Agent with Proactive Context Management</a>

*HF ★ 14 · Guangyi Liu, Gao Wu, Congxiao Liu… · [HF 镜像](https://huggingface.co/papers/2606.19926)*

针对MLLM驱动的移动GUI智能体在长周期任务中，ReAct式被动积累记录导致提示爆炸、关键跨应用信息丢失的问题，本文提出MemGUI-Agent：通过ConAct框架将上下文管理与UI操作同列为策略输出项，维护三类结构化上下文压缩提示规模，配套2956条标注轨迹的MemGUI-3K数据集训练，其8B版本在两项基准上获同参数开源最优性能。

<FeedbackButtons slug="item:arxiv-2606.19926" client:visible />

## <a href="https://arxiv.org/abs/2606.22875" target="_blank" rel="noopener noreferrer">FedOT: Ownership Verification and Leakage Tracing via Watermarks for Federated LDMs</a>

*HF ★ 7 · Wenlong Cheng, Yuan Gan, Yunqiu Xu… · [HF 镜像](https://huggingface.co/papers/2606.22875)*

针对联邦学习训练的隐扩散模型（LDM）易被恶意参与者泄露倒卖、现有水印方案无法溯源泄露源、且易通过更换VAE解码器擦除的问题，本文提出FedOT框架：设计分段水印分别用于所有权验证与恶意客户端溯源；新增隐向量变换绑定VAE与U-Net潜空间，换VAE会导致生成画质严重失效。实验证实其验权、溯源性能优异。

## <a href="https://arxiv.org/abs/2606.20670" target="_blank" rel="noopener noreferrer">Towards CSI-Native Foundation Models: A Channel-Adaptive Roadmap for 6G</a>

*Chenyu Zhang, Xinchen Lyu, Chenshan Ren…*

针对现有6G无线基础模型未将信道状态信息（CSI）视为传播响应、无法捕捉无线环境空时频固有几何特征的缺陷，本文提出信道自适应的CSI原生基础模型路线，构建统一框架对齐预训练等模块与三类信道约束。实验验证该方案多维度性能优于基线，导频开销仅7.01%，净频谱效率较现有最优方案提升15.5%，可支撑高效无线接入。

## <a href="https://arxiv.org/abs/2606.20673" target="_blank" rel="noopener noreferrer">NeuroShield: A Device-Agnostic Foundation Model for EEG Authentication</a>

*Matin Fallahi, Patricia Arias-Cabarcos, Thorsten Strufe*

针对现有EEG身份认证模型绑定训练时的采集设置、复用性差的痛点，本文提出设备无关的基础模型NeuroShield。其采用双阶段Transformer架构，可从可变通道、时长的EEG中提取身份判别嵌入，经1.5万余受试者数据预训练，微调后等错误率较SOTA低0.44-8.06个百分点，适配未见过的采集配置，已开源。

<FeedbackButtons slug="item:arxiv-2606.20673" client:visible />

## <a href="https://arxiv.org/abs/2606.20743" target="_blank" rel="noopener noreferrer">Massive Activations Are Architecturally Robust: A Controlled Scratch/Commitment Residual Stream Test</a>

*Maruthi Vemula (University of North Carolina at Chapel Hill)*

为验证Transformer中集中于序列起始token的大激活异常值，是残差流兼具读写双重角色的偶然产物还是功能必需，研究者提出将残差流拆分为可覆写计算流、只读解码累加器的Ledger残差架构。实验发现拆分后解码通道仍重构同类异常激活，稀疏惩罚反而强化其特征，证明大激活是有功能的架构稳健特征，而非偶然产物。
