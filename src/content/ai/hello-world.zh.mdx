## 为什么用 Astro

> 静态优先、岛屿架构、零默认 JS——对于一个内容驱动的个人站，这套组合的写作摩擦最低。

代码块测试：

```ts
function fib(n: number): number {
  if (n < 2) return n;
  return fib(n - 1) + fib(n - 2);
}
```

行内 `code` 也能 work。**加粗**、*斜体*、[链接](https://astro.build) 都自然渲染。
