# Hugging Face Daily Papers

## <a href="https://arxiv.org/abs/2606.12384" target="_blank" rel="noopener noreferrer">APPO: Agentic Procedural Policy Optimization</a>

*HF ★ 28 · Xucong Wang, Ziyu Ma, Yong Wang… · [HF 镜像](https://huggingface.co/papers/2606.12384)*

针对现有智能体强化学习工具调用方法依赖粗粒度单元做信用分配、无法准确定位关键中间决策的问题，本文发现关键决策点分散于全序列，单词元熵无法可靠反映其影响，据此提出APPO算法：用结合词元不确定性与后续收益的分支得分选探索点，搭配过程级优势缩放分配信用，在13个基准上较基线提升近4个点，兼顾调用效率与可解释性。

## <a href="https://arxiv.org/abs/2606.06036" target="_blank" rel="noopener noreferrer">Memory is Reconstructed, Not Retrieved: Graph Memory for LLM Agents</a>

*HF ★ 25 · Shuo Ji, Yibo Li, Bryan Hooi · [HF 镜像](https://huggingface.co/papers/2606.06036)*

本文针对大模型智能体长交互推理短板，以及现有记忆增强方案静态“先检索后推理”范式适配性差的问题，提出MRAgent框架：以关联记忆图存储记忆，将推理嵌入访问流程，主动迭代调整检索路径。在两个长时记忆基准上性能较基线最高提升23%，同时大幅降低token与耗时成本。

## <a href="https://arxiv.org/abs/2606.13707" target="_blank" rel="noopener noreferrer">Orchestra-o1: Omnimodal Agent Orchestration</a>

*HF ★ 23 · Fan Zhang, Vireo Zhang, Shengju Qian… · [HF 镜像](https://huggingface.co/papers/2606.13707)*

针对现有多智能体编排框架模态适配范围窄、难处理多模态异构交互复杂任务的问题，本文提出全模态智能体编排框架Orchestra-o1，搭载模态感知任务拆解、子代理在线专化、子任务并行执行机制，配套DA-GRPO强化学习方法训练的8B版本，在OmniGAIA基准准确率超次优方案10.3%，开源全模态智能体性能达当前最优。

*HF ★ 21 · Yongheng Zhang, Ziang Liu, Jiaxuan Zhu… · [HF 镜像](https://huggingface.co/papers/2606.14502)*

## <a href="https://arxiv.org/abs/2606.13432" target="_blank" rel="noopener noreferrer">OmniDirector: General Multi-Shot Camera Cloning without Cross-Paired Data</a>

*HF ★ 18 · Jiwen Liu, Shujuan Li, Zhixue Fang… · [HF 镜像](https://huggingface.co/papers/2606.13432)*

## <a href="https://arxiv.org/abs/2606.13705" target="_blank" rel="noopener noreferrer">Can Editing 1 Neuron Fix Repetition Loops in LLMs?</a>

*Aristotelis Lazaridis, Aman Sharma, Dylan Bates…*

针对Gemma系列指令微调模型在长事实列举任务中最高95%概率出现重复循环的问题，研究通过逐层消融、神经元归因定位到少量MLP神经元（大模型为路由专家），最小仅修改单个神经元权重即可消除重复且不影响通用性能，但该方法无法根除长推理时的“厄运循环”，后者本质是知识缺失问题，权重编辑无法补全缺失事实。

## <a href="https://arxiv.org/abs/2606.13740" target="_blank" rel="noopener noreferrer">Efficient On-Device Diffusion LLM Inference with Mobile NPU</a>

*Tuowei Wang, Yanfan Sun, Ju Ren*

扩散大语言模型（dLLM）可并行生成多token，适配端侧低延迟需求，但存在移动端推理算力开销高、移动NPU难利用的痛点。本文提出首个移动端NPU优化的dLLM推理框架llada.cpp，通过多块投机解码、双路渐进修正、交换优化内存运行时三项技术适配NPU特性，实测LLaDA-8B推理时延较CPU基线降17-42倍，且保留生成质量。
